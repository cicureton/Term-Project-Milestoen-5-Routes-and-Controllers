# Furniture Store Full Stack Node App

The full stack app for managing a furniture store using Node.js, Express, and SQLite.

## Installation
- Get the project
    - clone
        ```bash
        git clone https://github.com/YOUR_USERNAME/furniture-store-node.git
        ```
    - or download the zip.
- Open the project in VSCode.
- Open a Terminal.
- Install required packages.
    ```bash
    npm install
    ```
- Start the server.
    ```bash
    npm start
    ```
